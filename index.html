VUE Editable

Better editable element in Vue 2.x

Single line text Editable 

<editable value.sync='{{first}}'></editable>

Mutiple line text Editable 

<script>second = '{{second}}'</script>
<editable :value="{{second}}" type='textarea'></editable>

Number Editable 

<editable type='number' value={{third}}></editable>

Select Editable 

<script>options = {{JSON.stringify(options)}}</script>
<editable type='select' value={{forth}} :options='options'></editable>
value is {{forth}}

QUILL Editable 

<editable type='quill' value="{{fifth}}"></editable>
